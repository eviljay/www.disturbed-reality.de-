Linux/Unixam 31 Mrz 2005 03:44 pm

ubuntu 5.04 hoary

die meiner meinung nach beste linux distribution ubuntu kommt in einer neuen version. naja zumindest ist heute ein release candidate freigegeben worden
ubuntu basiert auf debian und liefert gnome 2.10 als desktop mit. wer lieber KDE will sollte sich das kubuntu projekt mal ansehen. ansonsten gibt es jedes halbe jahr eine neue version, jeweils mit aktuellster software. ich setzte die betas von ubuntu hier schon ne ganze weile ein und finde sie weit stabiler als das parallel installierte windows xp media center 2005.

bei der installation gab es keinerlei haken, als erstes(!!!) betriebssystem hat ubuntu (noch in der alten version warty 4.10) alle hardware erkannt. windows kotzt bei meinem pc eh immer total, wärend linux fast regelmäßig am sil-sata controller verzweifelt. naja bis auf dieses.&nbsp;
der gnome desktop wirkt wunderbar aufger&auml;umt, die bediehnung ist sehr intuitiv und die sachen sind dort wo sie sein sollen.&nbsp;
schaut euch am besten mal einige screenshots (golem) an, dann wisst ihr was ich meine.

ubuntu gibts als live-cd zum ausprobieren. live cd heisst cd brennnen, einlegen und davon booten. es wird nix installiert, aber man kann mal in linux reinschnuppern :)uub
