Verfügbare Aktionen

?action=browse
Zeigt die angegebene Seite an

?action=edit
Startet Bearbeitung der angegebenen Seite

?action=diff
Zeigt die Historie der aktuellen Seite an

?action=attr
Ruft einen Dialog zum Setzen der Passwörter (für z.B. Bearbeitung) auf

?action=postattr

?action=upload
Ruft einen Dialog zum Hochladen von Anhängen für die aktuelle Gruppe auf

?action=download&upname=datei.erw
Lädt die an die aktuelle Seite angehängte Datei datei.erw herunter

?action=crypt
Ruft einen Dialog auf, der ein Passwort so umwandelt, daß es direkt in der config.php eingesetzt werden kann

?action=print
Ruft die angegebene Seite mit der Stilvorlage (Skin) auf, welches in $ActionSkin['print'] angegeben wurde

?action=search
Zeigt die Suchmaske auf der aktuellen Seite an

?action=search&q=suchbegriff
Führt die Suche nach suchbegriff durch und zeigt das Ergebnis auf der aktuellen Seite an

?action=source
Zeigt den Seitenquelltext an

?action=logout
Löscht Informationen zum Autor, Passwort und Login

?setprefs=BeliebigeGruppe.EigeneEinstellungen
Setzt Cookie mit Verweis zu eigenen Voreinstellungen. Siehe Site Voreinstellungen?

Durch $EnableDiag aktivierte Aktionen:

Nachfolgende Aktionen sind nur verfügbar, wenn $EnableDiag = 1 in der config.php - Datei eintragen ist. Sie können für die Fehlersuche verwendet werden und sollten nicht als Standard für den normalen Betrieb eingesetzt werden.

?action=ruleset
Zeigt die Liste aller Markups in 3 Spalten an:
  • column 1 = Name des Markups (1. Parameter von Markup() )
  • column 2 = Wann trifft das Markup zu (2. Parameter von Markup() )
  • column 3 = PmWiki's interner Sortierschlüssel (hergeleitet von #2)
(siehe auch Eigene Auszeichnungen?)

?action=phpinfo
Zeigt die Informationen von phpinfo() an und endet dann. Es wird keine Seite verarbeitet

?action=diag
Zeigt die Werte aller globalen Variablen an und endet dann. Es wird keine Seite bearbeitet
