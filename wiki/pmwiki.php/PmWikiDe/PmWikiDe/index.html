PmWikiDe / Übersicht

(deutsche Übersetzung der Seite PmWiki.PmWiki, Stand 09.09.2005)

PmWiki ist ein wiki-basierendes System für die gemeinsame Erstellung und Wartung von Webseiten. PmWiki Seiten sehen genauso aus wie normale Webseiten, bis auf den kleinen Unterschied, dass sie einen "Bearbeiten" Link haben. Damit ist es einfach bestehende Seiten zu verändern bzw. neue Seiten zu erstellen. Hierzu gibt es einige Grundlagen. Man muss nichts über HTML oder CSS wissen bzw. es benutzen. Das Bearbeiten von kann allgemein möglich sein, es kann aber auch auf nur eine kleine Gruppe von Authoren beschränkt werden.

Laden Sie die neueste Version von PmWiki als ZIP-Archiv (pmwiki-latest.zip(Links überprüfen)) oder als GZIPtes TAR-Archiv (pmwiki-latest.tgz(Links überprüfen)) von http://www.pmwiki.org/pub/pmwiki/(Links überprüfen) herunter.

Machen Sie dann bei der Installation weiter.

PmWiki wurde in PHP geschrieben und wird unter den Bedingungen der "General Public License" vertrieben. Es lässt sich einfach installieren, für eigene Bedürfnisse anpassen und pflegen, und das für eine Vielzahl von Anwendungen. Diese Webseite arbeite mit der Version pmwiki-2.1.5.

PmWiki ist ein eingetragenes Markenzeichen von Patrick R. Michaud(Links überprüfen).

Die PmWiki Homepage findet sich im Internet unter pmwiki.org(Links überprüfen).
