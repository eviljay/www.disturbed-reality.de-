PmWikiDe / Variablen

Administratoren (Grundlagen) (deutsche Übersetzung der Seite PmWiki.Variables, Stand 12.09.2005)

Diese Seite dokumentiert die in PmWiki verfügbaren Variablen für Lokale Anpassungen. Vieles in dieser Dokumentation ist immer noch unvollständig, aber es wird daran gearbeitet. Gerne können auf der englischen Orignalseite Platzhalter für Variablen eingetragen werden, wenn man diese Variablen beschrieben haben möchte und nicht weiß was sie tun.

Verwechseln Sie nicht diese Variablen (die nur in PHP gesetzt und verwendet werden) mit den Variablen zur Auszeichnung?.

Die Beschreibung ist auf mehrere Seiten aufgeteilt

  • Grundlegende Variablen - Haupt-Variablen
  • Layout Variablen? - Das Seitenlayout kontrollierende Variablen
  • Pfad Variablen? - Variablen, die benutzt werden, um verschiedene Dateipfade auf dem Server anzugeben
  • Variablen zu Verweisen? - Die Darstellung von Verweisen in Seiten kontrollierenden Variablen
  • Variablen für die Bearbeitung? - Variablen die für das Bearbeiten benötigt werden
  • Variablen für das Hochladen - Variablen für das Hochladen von Dateien/Anhängen
  • i18n Variablen? - Variablen welche für die Internationalisierung benötigt werden (i18n=Abkürzung für internationalization)
  • Sonstige Variablen? - Variablen die noch nicht klassifiziert worden sind
  • Variablen für die Benachrichtigung per E-Mail? - Mails die bei Seitenänderungen geschickt werden

Eine komplette Übersicht der beschriebener Variablen findet man nachfolgend auf dieser Seite.

Im Allgmeinen wird der Wert von Variablen, deren Namen auf 'Fmt' enden (beispielsweise $PageLayoutFmt), für globale $-Variablenersetzungen vor der Ausgabe verarbeitet. Es werden also Zeichenketten wie $Name und $PageUrl mit dem Namen und der Internateadresse der Seite ersetzt, wenn die Zeichenkette gedruckt wird.
